# 墨水屏配置小程序

基于蓝牙连接的墨水屏配置工具，支持 WiFi 配置和参数设置。

## 功能特性

### 📱 蓝牙连接
- 自动搜索附近设备
- 实时连接状态显示
- 支持 BLE 低功耗蓝牙
- 自动重连机制

### 📶 WiFi 配置
- WiFi 扫描功能
- SSID 和密码输入
- 密码可见性切换
- 配置保存到本地和设备

### ⚙️ 参数设置
- 天气来源配置（实时/每日）
- 默认城市设置
- 周信息显示开关
- 金价显示开关
- 标签天数设置
- 冲刺日期配置
- 日程提醒设置

## 使用方法

### 1. 连接设备
- 打开小程序，确保手机蓝牙已开启
- 点击「搜索设备」按钮
- 选择「WuJinOS」设备进行连接

### 2. 配置 WiFi
- 连接成功后，切换到「WiFi」标签
- 输入 WiFi SSID 和密码
- 或点击「扫描WiFi」自动获取附近网络
- 点击「保存配置」发送到设备

### 3. 参数设置
- 切换到「设置」标签
- 根据需求配置各项参数
- 点击「保存配置」应用设置
- 可点击「恢复默认」重置所有配置

## 技术实现

### 蓝牙通信协议
```
发送格式: JSON 字符串
{
  "cmd": "命令名称",
  "data": { ...参数 }
}

接收格式: JSON 字符串
{
  "cmd": "命令_result",
  "data": { ...返回数据 }
}
```

### 支持的命令
| 命令 | 说明 |
|------|------|
| `scan_wifi` | 扫描附近 WiFi |
| `set_wifi` | 设置 WiFi 配置 |
| `set_params` | 设置参数配置 |
| `reset_params` | 恢复默认配置 |

### BLE UUID
- Service UUID: `0000FFE0-0000-1000-8000-00805F9B34FB`
- Characteristic UUID: `0000FFE1-0000-1000-8000-00805F9B34FB`

### 部署步骤
1. 使用微信开发者工具打开项目
2. 修改 `project.config.json` 中的 `appid`
3. 上传代码到微信后台
4. 提交审核发布

## 注意事项

1. **蓝牙权限**：首次使用需要授权蓝牙权限
2. **设备兼容**：确保墨水屏支持 BLE 蓝牙
3. **连接距离**：建议距离不超过 3 米
4. **数据安全**：WiFi 密码仅保存在设备本地
5. **超时处理**：所有操作默认 10 秒超时

## 常见问题

### Q: 搜索不到设备？
A: 请检查墨水屏是否开机，蓝牙是否已开启。

### Q: 连接失败？
A: 请重试连接，或检查是否有其他设备已连接。

### Q: 配置保存失败？
A: 确认设备已连接，网络稳定后重试。

### Q: WiFi 连接不上？
A: 检查密码是否正确，路由器是否支持 2.4GHz。
